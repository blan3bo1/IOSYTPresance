=== iOS YouTube Presence ===

Discord Rich Presence for YouTube iOS

INJECTION INSTRUCTIONS:

1. Download YouTube IPA (decrypted)
2. Use eSign, Azula, or similar tool
3. Inject IOSYTPresence.dylib into the YouTube IPA
4. Sign with your certificate
5. Install on device

FEATURES:
- Shows current YouTube video in Discord
- Displays channel name and progress
- Updates play/pause status
- No jailbreak required

REQUIREMENTS:
- iOS 13.0 or later
- YouTube app (decrypted IPA)
- Discord app
- Signing service (free or paid)

NOTES:
- Works with sideloading (7-day limit for free accounts)
- Requires Discord running in background
- Video detection starts automatically
